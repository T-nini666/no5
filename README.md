# 数据可视化第5章 - 交互式可视化网站

## 项目简介

本项目将第5章中的matplotlib可视化代码转换为交互式网页图表，使用Plotly.js库实现动态、可交互的数据可视化。

## 功能特点

### 1. 产品销售额分析
- **交互功能**：
  - 切换图表类型（折线图/柱状图）
  - 显示/隐藏数值标注
- **数据内容**：
  - 产品A与产品B全年12个月的销售额对比
  - 原始matplotlib代码中的完整数据展示

### 2. 养猫养狗人群比例分析
- **交互功能**：
  - 国家筛选（全部/亚洲/欧美）
- **数据内容**：
  - 12个国家养猫和养狗人群的比例对比
  - 水平条形图展示，便于比较

### 3. 抖音用户分析
- **交互功能**：
  - 年份选择（2017年/2018年/同时显示）
- **数据内容**：
  - 抖音用户在不同城市的增长倍数
  - 2017年和2018年用户地区分布比例
  - 多子图布局展示

### 4. 气温与降水量关系分析
- **交互功能**：
  - 数据类型筛选（全部/仅气温/仅水量）
- **数据内容**：
  - 某地区全年平均气温变化
  - 降水量和蒸发量的对比
  - 双Y轴图表展示

### 5. 子图布局示例
- **交互功能**：
  - 切换布局类型（基本子图/自定义布局/共享坐标轴）
- **展示内容**：
  - 多种子图布局方式演示
  - 共享坐标轴功能展示

## 技术栈

- **前端框架**：纯HTML + CSS + JavaScript
- **可视化库**：Plotly.js（功能强大的交互式图表库）
- **响应式设计**：支持桌面和移动设备
- **样式设计**：现代渐变背景，毛玻璃效果

## 使用说明

### 1. 直接打开使用
- 双击 `index.html` 文件即可在浏览器中打开
- 推荐使用Chrome、Firefox或Edge等现代浏览器

### 2. 通过本地服务器运行（推荐）
```bash
# 如果使用Python 3
python -m http.server 8000

# 如果使用Python 2
python -m SimpleHTTPServer 8000

# 如果使用Node.js
npx http-server
```
然后访问 `http://localhost:8000`

### 3. 交互操作
1. 点击顶部导航按钮切换不同可视化案例
2. 每个案例都提供交互控件（下拉菜单、复选框等）
3. 图表支持悬停查看详细信息
4. 支持缩放、平移等交互操作
